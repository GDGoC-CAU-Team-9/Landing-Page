# SafePlate Landing Page (Next.js)

SafePlate Flutter 앱 소개용 랜딩페이지입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

이후 `http://localhost:3000` 접속

## 검증

```bash
npm run lint
npm run build
```

## Vercel 배포

1. GitHub 저장소를 Vercel에 Import
2. Framework Preset은 `Next.js` (자동 인식)
3. Build Command: `next build` (기본값)
4. Output Directory: 기본값 사용
5. Deploy
